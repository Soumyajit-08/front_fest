## 📌 Project Overview

Earth's Exhale is a static frontend website designed for a plant store. It provides a clean and user-friendly interface where users can browse featured plants, explore best-selling products, and read customer testimonials.

This project demonstrates responsive web design principles, structured layouts, and modern UI styling.

## Features

- Responsive landing page
- Modern navigation bar
- Hero/Banner section
- Featured indoor plant section
- Top selling plants showcase
- Customer review section
- Best O₂ plants section
- Attractive product cards
- Font Awesome icons
- Clean and organized UI

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Responsive Design

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Soumyajit-08/front_fest.git
```

### 2. Open the project

```bash
cd front_fest
```

### 3. Run the project

Simply open `index.html` in your browser.

Or, if you're using VS Code:

- Install the **Live Server** extension.
- Right-click on `index.html`.
- Click **Open with Live Server**.
